# 专题 Prompt 集合

按场景选择下列模板。

## A. 翻译单函数

```text
翻译 `../tinyexpr-c/tinyexpr.c` 中的 <函数范围>。
先做：Python 类比解释 → 翻译决策 → 风险点 → Rust 代码。
仅处理该函数，并附 1-2 个最小测试建议。
```

## B. 只讲解不翻译

```text
请解释 `../tinyexpr-c/tinyexpr.c:<行号>`。
仅做概念讲解，使用 Python 类比，不输出翻译代码。
```

## C. 测试失败诊断

```text
以下测试失败：
<错误输出>
先给 3 个可能原因及验证方法，不要直接改代码。等我确认方向后再修改。
```

## D. 架构决策讨论

```text
讨论 <架构问题>。请给至少 3 种方案、优缺点和适用场景，最后给推荐并等待确认。
```

## E. 对照不一致分析

```text
`../scripts/compare.ps1` 存在差异：
<失败用例>
请区分翻译错误、库实现差异、浮点容差问题，并给验证步骤与处理建议。
```
